# Bus vol Bikes app

## Starten

```bash
npm install
npm start
```

## URLs

- `/voorraad`
- `/api/fietsen`
- `/api/fietsen.xml`
- `/api/health`
- `/fiets/:id`

## WordPress embed

```html
<div id="bvb-voorraad"></div>
<script>
window.BVB_API_BASE = 'https://api.busvolbikes.nl';
</script>
<script src="https://api.busvolbikes.nl/embed.js"></script>
```
